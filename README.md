A collection of end-to-end data science and machine learning projects completed as part of my Post Graduate Program in Data Science & Business Analytics. Each project covers the full pipeline — problem framing, data exploration, modelling, and business interpretation of results.

## Projects

### 1. Machine Learning
**Files:** `Machine Learning project 1 - Final.ipynb` · `Machine Learning Project Q2.ipynb`

Applied supervised learning techniques to real-world datasets. Covers feature engineering, model selection, hyperparameter tuning, and evaluation using accuracy, precision, recall, and F1-score.

**Techniques used:** Regression · Classification · Decision Trees · Random Forest · Cross-validation

---

### 2. Predictive Modelling
**Files:** `Predictive Modelling Question 1.ipynb` · `Predictive Modelling Question 2.ipynb`

Built predictive models to forecast outcomes from structured data. Focus on comparing multiple algorithms and selecting the best performer based on evaluation metrics.

**Techniques used:** Linear & Logistic Regression · Model comparison · ROC-AUC · Confusion matrix

---

### 3. Data Mining
**Files:** `Data Mining Final Project Question 1.ipynb` · `Data Mining Project Question 2.ipynb`

Explored large datasets to extract patterns and actionable insights. Applied clustering and association techniques alongside supervised learning.

**Techniques used:** K-Means Clustering · Association Rules · EDA · Data preprocessing

---

### 4. Time Series Analysis
**Files:** `Time Series - Sparkling.ipynb`

Analysed time-ordered data to identify trends, seasonality, and forecast future values. Applied to a real business dataset.

**Techniques used:** ARIMA · Decomposition · Stationarity testing (ADF) · Forecasting

---

### 5. Statistical Methods for Decision Making (SMDM)
**Files:** `Project 2 SMDM.ipynb` · `GL SMDM Project.pdf`

Applied statistical inference and hypothesis testing to support data-driven decision making in business contexts.

**Techniques used:** Hypothesis testing · ANOVA · Chi-square · Confidence intervals · Probability distributions

---

### 6. Advanced Statistics
**Files:** `Project AS.ipynb`

Deep dive into statistical modelling covering probability theory, distributions, and inferential statistics applied to real datasets.

**Techniques used:** Descriptive statistics · Regression analysis · Distribution fitting · Statistical inference

---

### 7. Algorithmic Trading (Financial Risk Analytics)
**Files:** `FRA_Week-2_Algorithmic_Trading.ipynb`

Explored algorithmic trading strategies using Python. Applied quantitative finance concepts to analyse price data and build simple rule-based trading systems.

**Techniques used:** Moving averages · Backtesting · Returns analysis · Risk metrics
